# McBride Creative Website - Quick Start Guide

### 1. Install WordPress (5 minutes)
- Use your hosting provider's one-click WordPress installer
- Site Name: **McBride Creative**
- Admin Email: **hello@mcbridecreative.com**

### 2. Install Theme (2 minutes)
- WordPress Admin > Appearance > Themes > Add New > Upload
- Upload `mcbride-creative-theme.zip`

### 3. Install Essential Plugins (5 minutes)
Install these 3 must-have plugins:
1. **WPForms Lite** - Contact forms
2. **Yoast SEO** - Search optimization
3. **Wordfence** - Security

### 4. Create Key Pages (10 minutes)

**Homepage:**
- Pages > Add New > Title: "Home"
- Template: "Homepage"
- Publish
- Settings > Reading > Static page: "Home"

**Portfolio:**
- Pages > Add New > Title: "Portfolio"
- Add portfolio grid
- Publish

**Services:**
- Pages > Add New > Title: "Services"
- Add 3 packages (Per-Show, Season, Premium)
- Publish

**About:**
- Pages > Add New > Title: "About"
- Add your story + headshot
- Publish

**Contact:**
- Pages > Add New > Title: "Contact"
- Add WPForms contact form
- Add email: hello@mcbridecreative.com
- Publish

### 5. Add Portfolio Items (5 minutes)
- Portfolio > Add New
- Add 3-5 MCG projects
- Include images and descriptions
- Publish each

### 6. Configure Settings (3 minutes)
- **Logo:** Appearance > Customize > Site Identity
- **Menus:** Appearance > Menus > Add pages to Main Navigation
- **Theme Options:** Customize > McBride Creative Options
  - Contact email
  - Location
  - Instagram handle

## ðŸŽ¨ Content Guidelines

### Writing Tone
- Professional but approachable
- Confident without arrogance
- Passionate about theatre
- Results-oriented
- Trustworthy for volunteer boards

### Image Specs
- **Portfolio:** 1200x800px minimum
- **Hero Images:** 1920x1080px
- **Logos:** 400x100px (transparent PNG)
- **Favicon:** 512x512px
- **Format:** JPG for photos, PNG for graphics
- **Optimize:** Keep under 200KB each

### SEO Best Practices
- **Title Tags:** Under 60 characters
- **Meta Descriptions:** 150-160 characters
- **Headings:** One H1 per page, use H2-H4 hierarchy
- **Alt Text:** Describe all images
- **Internal Links:** Link between pages

## ðŸ†˜ Quick Troubleshooting

### Can't log in?
- Use password reset: yourdomain.com/wp-login.php?action=lostpassword

### Contact form not sending?
- WPForms > Settings > Email
- Check "Send To" address
- Install WP Mail SMTP if needed

### Images not uploading?
- Check file size (max 10MB usually)
- Check file format (JPG, PNG, GIF, WebP)
- Check hosting PHP upload limit

### Site looks broken?
- Clear cache (WP Rocket > Clear Cache)
- Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
- Deactivate plugins one-by-one to find conflict

### Slow loading?
- Optimize images (Smush plugin)
- Enable caching (WP Rocket)
- Check hosting performance
- Remove unused plugins
